Detailed Description

ContactsWidget::ContactsWidget ( QWidget *  parent = nullptr)
explicit

Konstruktor klasy ContactsWidget.

Parameters
Wskaznikna rodzica Widgetu domyślnie ustawiony na NULL

W konstruktorze wyświetlane jest okno UI, które pokazuje wszystkie kontaktu aktualnie zalogowanego użytkownika. Tworzy połączenie przycisków dodających kontakt i edytujących kontakt, z funkcją, która odświeża dane w tabeli. Następnie tworzy tabelę z polami "ID", "Imie", "Nazwisko", "Numer telefonu", "Email", "Adres".
